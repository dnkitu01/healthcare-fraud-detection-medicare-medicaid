# Medicare & Medicaid Fraud Detection Using Machine Learning

## Project Overview

Healthcare fraud costs billions of dollars annually and impacts healthcare organizations, insurers, and government programs. This project explores how machine learning can be used to identify patterns associated with potentially fraudulent healthcare claims and provider behavior.

## Business Problem

Fraudulent claims increase healthcare costs and reduce operational efficiency. The goal of this project was to develop predictive models capable of identifying potentially fraudulent cases using Medicare and Medicaid healthcare data.

## Dataset

The project integrates multiple healthcare datasets including:

- Beneficiary Information
- Inpatient Claims
- Outpatient Claims

## Methodology

### Data Preparation

- Data Cleaning
- Feature Engineering
- Missing Value Treatment
- Dataset Integration
- Removal of Data Leakage Variables

### Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Model Evaluation

Performance was evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score

## Key Findings

- Random Forest outperformed Logistic Regression.
- Feature engineering improved predictive performance.
- Removing leakage variables, improved model reliability and real-world applicability.
